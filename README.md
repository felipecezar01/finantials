### ✨ Key Features

* **🇧🇷 Brazilian Context:** Logic built entirely around **monthly cycles** rather than daily/annual budgets.
* **💳 Installment Management:** Smart handling of credit card installments ("Parcelado") across future months.
* **📝 Full Control:** Add, edit, or delete any transaction easily.
* **📊 Visual Analytics:** Interactive charts for both **Income** and **Expenses** to track cash flow.
* **🔒 Offline-First:** Powered by **Hive (NoSQL)**. Your data never leaves your device.
* **🌑 Dark Mode:** Immersive UI designed for daily usage.

### 🛠️ Tech Stack

* **UI Framework:** Flutter
* **Database:** Hive (Key-value database)
* **State Management:** Native `ValueListenableBuilder`
* **Charts:** FL Chart
